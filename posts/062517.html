What is .matches?

Element.matches(slectorString)

The .matches function is part of the Element class prototype. This means it can be called an any part of the DOM or any variable that has an element of the DOM stored. It is used to test for truthiness between the given element and the selectorString arguement. This can be much easier than using the typical comparison operator.

Problems with .matches

According to MDN, this is the method that should be used as a standard. But, as with all things JavaScript, you can never be certain as to who is using it properly. To circumvent this issue the folloing function can be used as a catch all to guaruntee you will not run into an issue of having your program broken. (at least not because of this method)

Function

function matchesSelector(el, selector) { var p = Element.prototype; var f = p.matches || p.webkitMatchesSelector || p.mozMatchesSelector || p.msMatchesSelector || function(s) { return [].indexOf.call(document.querySelectorAll(s), this) !== -1; }; return f.call(el, selector); }

Examples

var test = document.getElementbyId('container') test.matches('#container') matchesSelector(test, '#container')
